## 1. Personal Resume/CV

**Covers:** Headings, paragraphs, lists, links, text formatting, tables

- Contact information with email links
- Work experience in tables
- Skills in unordered lists
- Education timeline in ordered lists

## 2. Recipe Collection Website

**Covers:** Multiple pages, navigation, images, nested lists

- Homepage linking to 5+ recipes
- Ingredients (unordered lists) and steps (ordered lists)
- Images with alt text and dimensions
- Preparation time and servings info

## 3. Multi-Page Blog

**Covers:** Semantic HTML5, article structure, time elements

- Homepage with article previews
- Individual blog post pages using header, nav, article, aside, footer
- Author information and publish dates
- Related posts sidebar

## 4. Comprehensive Registration Form

**Covers:** All form elements, validation, fieldsets

- Personal info (text, email, tel, date inputs)
- Preferences (radio, checkbox, select, datalist)
- Password with pattern validation
- Terms acceptance with required attribute
- Grouped sections with fieldsets

## 5. Product Catalog

**Covers:** Advanced tables, figures, definition lists

- Product comparison table with colspan/rowspan
- Product specifications in definition lists
- Multiple product images with figure/figcaption
- Table headers (thead) and body sections

## 6. Video/Audio Portfolio

**Covers:** Media elements, embedded content, tracks

- Video showcase with multiple formats
- Audio player with source fallbacks
- Subtitle/caption tracks (WebVTT)
- Embedded iframe content
- Canvas element placeholder

## 7. Interactive FAQ Documentation

**Covers:** Details/summary, dialog, semantic structure

- Collapsible FAQ sections with details/summary
- Modal information using dialog element
- Deep heading hierarchy (h1-h6)
- Table of contents with anchor links
- Code examples in pre/code tags

## 8. Accessible Contact Portal

**Covers:** ARIA, accessibility, advanced forms

- Complex form with ARIA labels and descriptions
- Error message associations
- Skip navigation links
- Landmark roles (main, complementary)
- Focus management structure

## 9. Responsive Image Gallery

**Covers:** Picture element, srcset, performance

- Picture element for art direction
- Srcset and sizes for responsive images
- Lazy loading attributes
- Resource hints (preload, prefetch)
- Multiple image format fallbacks

## 10. Multilingual Structured Data Site

**Covers:** Internationalization, microdata, metadata

- Multiple language sections with lang attributes
- Schema.org microdata markup
- Open Graph and Twitter Card meta tags
- JSON-LD structured data
- RTL/LTR text direction handling
- Manifest.json for PWA basics

---

**Progression Tips:**

- Build projects 1-4 for **Beginner** level mastery
- Projects 5-7 for **Intermediate** level
- Projects 8-10 for **Advanced/Expert** level
